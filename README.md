# Inventario_de_Produtos
# Introdução
Um sistema de inventário de produtos feito utilizando python e sua biblioteca de banco de dados, o sqlite3.
# Funções desse sistema:
• Adicionar Produtos

• Listar Todos os Produtos

• Buscar Produtos

• Atualizar Informações

• Remover Produtos

• Ver Valor Total do Estoque
